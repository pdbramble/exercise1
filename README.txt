These files solve the following problem:

Write some code that will accept an amount and convert it to the
appropriate string representation.
Example:
Convert 2523.04
to "Two thousand five hundred twenty-three and 04/100 dollars"




The files include:

com/paul/ex1/CurrencyTranslator.java
com/paul/ex1/InputToken.java
com/paul/ex1/TripleDigitToken.java

You may run these from the junit file -

com/paul/ex1/CurrencyTranslatorTests.java
